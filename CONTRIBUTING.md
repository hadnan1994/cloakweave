# Contributing to Cloakweave

Thanks for helping improve Cloakweave.

## Development

Install dependencies:

```bash
npm install
```

Start the desktop app:

```bash
npm run dev
```

Run tests:

```bash
npm test
```

Run type checks:

```bash
npm run typecheck
```

## Good first areas

- File parser improvements
- UI polish
- Local model integrations
- Retrieval quality improvements
- Tests
- Documentation

## Pull requests

- Keep changes focused.
- Add or update tests for behavior changes.
- Avoid adding telemetry or cloud calls by default.
- Be clear when a feature can send document content outside the local machine.
